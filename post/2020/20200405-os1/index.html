docker安装xv6环境

我使用docker进行环境搭建，构建自己的image，方便部署，之后换电脑也无需重复安装各种工具。 我的image: docker pull zuojxin/xv6

docker 简单命令

  • 创建自己的docker：docker build -t /:
  • 安装镜像：docker pull
  • 删除镜像：docker rmi
  • 删除容器：docker rm
  • 查看已安装镜像：docker images
  • 启动镜像容器： docker run -t -i
  • 保存容器：docker commit
  • 上传镜像：docker push
  • 本地文件拷贝到docker容器：docker cp 文件 :容器内路径
  • 精简image大小：http://www.dockerinfo.net/3328.html
  • 清理docker缓存：https://blog.fundebug.com/2018/01/10/how-to-clean-docker-disk/
